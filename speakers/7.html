Kwen Peterson

Bio:

Kwen is a senior software developer for Raven Industries. He has over a decade of experience working in the Sioux Falls area, primarily in the .NET space, and is an organizer for the Sioux Falls Developers Group and South Dakota Code Camp.

Twitter: kwenarik

Company: Raven Industries

Session(s):

  • .NET Core or .NET Framework: Which one should I choose?

    Does the release of .Net Core mean the classic .NET Framework is deprecated? Going forward are all new features only going to be available in Core? I just do Windows development and never plan on deploying to Linux, why should I care about .Net Core? What does this mean for ASP.NET and Entity Framework? Does this new .NET Standard 2.0 thing play into this somehow?

    We'll look at answering these questions (as well as others) during this talk and lay out what the roadmap is long-term for .NET as well as looking at some code examples and performance comparisons.
